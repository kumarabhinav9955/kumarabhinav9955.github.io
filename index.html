Password_Manager

This password manager app, built with React, Vite, Tailwind CSS, and Lord Icons, securely stores passwords in the user's localStorage, ensuring privacy by keeping data on the device. Users can easily add, update, and delete password entries, with data persisting across sessions. Since it uses localStorage, the data is not sent to external servers, ensuring no third-party access. However, localStorage has a 5MB limit and doesn’t sync across devices. The app is simple and intuitive, prioritizing user control over security. While encryption could improve security, and cross-device syncing could enhance convenience, the app’s privacy-first approach makes it a great solution for users seeking to securely store passwords locally without relying on cloud storage. Future updates could include additional features such as data encryption and sync options, further improving security and user experience.

KeyLogger-Capturing-key-stokes

This Python program implements a keylogger, which is a type of software that records the keystrokes made by a user on their computer. It utilizes the pynput library to monitor and capture key events. The keylogger differentiated between the key press, key hold, and key release events..
